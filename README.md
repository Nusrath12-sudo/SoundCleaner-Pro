# SoundCleaner Pro 🎵

Remove background noise from audio and download cleaned mp3.

## Features
- Upload any mp3/wav
- AI Noise Reduction with noisereduce
- Play cleaned audio in browser
- Download as hello_clean.mp3
- Bass control slider

## How to Run Locally
1. Install Python 3
2. Clone repo: `git clone https://github.com/Nusrath12-sudo/SoundCleaner-Pro.git`
3. Run: `pip install flask librosa noisereduce soundfile`
4. Run: `python app.py`
5. Open `http://127.0.0.1:5000`
